🚀 PASOS PARA HACER DEPLOY EN VERCEL

1️⃣  IR A VERCEL
    → https://vercel.com/new

2️⃣  CONTINUAR CON GITHUB
    → Click "Continue with GitHub"
    → Autoriza a Vercel (primera vez)

3️⃣  IMPORTAR REPOSITORIO
    → Busca: viruzchuleno-alt/CLIENTELA
    → Click "Import"

4️⃣  CONFIGURAR (dejar todo por defecto)
    Project Name: clientela
    Framework: Other
    Root Directory: ./
    
    → Click "Deploy"

    → Vercel construye y despliega
    → ¡Listo!
